bool getModelVisibilityOnCreation ()
 Get the Model Visibility On Creation object. This is used to optimize the loading of objects in the scene. By default, every addition will require the rendering engine to update the BVH, and that takes time. By making the model invisible, it is created but not added to the scene. The set_models_visibility function can be used to make all models visible at once.
