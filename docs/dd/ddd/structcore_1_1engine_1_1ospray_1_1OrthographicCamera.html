Detailed Description

Implements a straightforward orthographic camera for orthographic projections, without support for Depth of Field or Motion Blur

A simple orthographic camera. This camera type is loaded by passing the type string "orthographic" to ospNewCamera

The orthographic camera supports the following parameters 

vec3f(a) pos;    // camera position
vec3f(a) dir;    // camera direction
vec3f(a) up;     // up vector
float    height; // size of the camera's image plane in y, in world
coordinates
float    aspect; // aspect ratio (x/y)
