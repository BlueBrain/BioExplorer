class  bioexplorer::rendering::VectorFieldsRenderer
 The VectorFieldsRenderer class allows visualization of magnetic fields created by atoms in the 3D scene. An Octree acceleration structure has to be built by the be-build-fields API in order to feed the renderer with the information needed to compute the value of the field for every point in the 3D space.
